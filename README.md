# The Demon Core

A two-track project: (1) a deep-dive history of the **Los Alamos plutonium core** that killed Harry Daghlian (1945) and Louis Slotin (1946) and would have been the fissile pit of the third atomic bomb intended for Japan; (2) a working **replica build** of the core itself (3D-printed plutonium sphere, ~89 mm / 6.2 kg analog) and its Manhattan-Project carrying case (the magnesium "plutonium box" designed by Philip Morrison and machined by Ralph Sparks, rebuilt here in the style of Adam Savage's 2024 *Tested* reproduction).

Layout conventions live in `../`'s CLAUDE.md and the Electronics umbrella's `_shared/structure.md`. See `.claude/CLAUDE.md` for project-specific direction.
